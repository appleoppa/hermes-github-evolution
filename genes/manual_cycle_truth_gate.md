# 手动开智进化循环入口门禁

本门禁来自一次手动真实闭环修正：外部阅读证据不足时，必须即时补读真实来源后再收束。

## 下次入口必须先检查

1. 不能用文件存在、状态字段、总结文字证明周期完成。
2. 每回合必须有：公式代入、EVM缺陷映射、河图洛书判断链、外部原文阅读、行为门禁变化。
3. 外部学习不能只有搜索结果；必须能证明读到原文内容，并提炼为下一步动作限制。
4. 评分器必须先审自身：不能手写修复后分数，不能用同一治理系数重复自证。
5. 任一回合证据不足时，状态只能是“证据不足”，必须补足后才能收束。

## 本周期形成的新门禁

每回合结束前必须回答：这个学习点具体改变了下一回合入口什么动作？如果没有改变入口，就不算完成学习。
