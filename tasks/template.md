# GitHub 进化闭环任务模板

任务名：顶级项目进化基因提炼

## 输入

- 主题：智能体自我进化、自动化验证、代码生成、知识回流
- 主要调度：MiniMax
- 最强主脑：GPT-5.5（进化主导）
- 辅助脑：DeepSeek 4

## 执行步骤

1. 搜索 GitHub 顶级项目
2. 读取项目说明、架构、示例和测试方式
3. 提炼可复用机制，不复制无关代码
4. 生成进化基因卡片
5. 在容器或远程流水线中做最小验证
6. 将结果写回 inbox，等待本地 Hermes 吸收

## 验收标准

- 有明确来源
- 有可验证机制
- 有适配 Hermes 的落地方式
- 有失败边界说明
- 不把搜索结果冒充已吸收能力
